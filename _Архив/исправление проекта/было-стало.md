# Стало:

Приведите названия в соответствие с хорошим стилем:
- несколько слов в названии запишите в «змеином_регистре»,
- все символы сделайте строчными.
- устраните пробелы.

Для этого переименуйте колонки так:
- `'userID'` → `'user_id'`;
- `'Track'` → `'track'`;
- `'City'` → `'city'`;
- `'Day'` → `'day'`.
